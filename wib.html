Wikipedia Integrated Browser (Mai 2017)

J'ai réalisé le WIB dans le cadre de mon stage de 2 mois au Laboratoire Informatique de Grenoble.
Cette page web a pour objectif de permettre au public de tester un nouveau moteur de recherche (cf: Écho)
et pour les chercheurs (équipe AMA) de le comparer à d'autres approches classiques.
La page web est reliée à une base de données SQL (que j'ai réalisé) qui engistrera de façon anonyme les actions de l'utilisateur,
ce qui permet au chercheur de comparer les différents moteurs.
Bien que la page manque d'esthetisme, elle est fonctionnelle et permet d'interagir avec le moteur de recherche,
en sélectionnant les suggestions de termes, catégories et documents. Disponible ici: WIB
